# School Health Trends Analysis

## 📌 Overview
This project analyses monthly illness data (flu, fever, asthma, stomach‑related cases) from a school environment to identify seasonal health patterns. The goal is to demonstrate how data analytics can support preventive health planning.

## 📊 Dataset
The dataset includes 12 months of anonymised illness counts.

Columns:
- Month  
- Flu  
- Fever  
- Asthma  
- Stomach Upset  

## 🧪 Methods
- Data cleaning and structuring  
- Trend analysis  
- Visualisation using bar and pie charts  
- Python libraries: Pandas, Matplotlib  

## 📈 Key Insights
- Respiratory illnesses peak in winter (Dec–Jan)  
- Asthma increases in late spring and summer  
- Stomach‑related illnesses remain steady  

## 🖼 Visuals
Bar and pie charts are available in the `visuals/` folder.
