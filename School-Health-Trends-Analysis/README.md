# School Health Trends Analysis

## 📌 Overview
This project analyses monthly illness data (flu, fever, asthma, stomach-related cases) from a school environment to identify seasonal health patterns.

## 📊 Dataset
Columns:
- Month  
- Flu  
- Fever  
- Asthma  
- Stomach Upset  

## 🧪 Methods
- Data cleaning  
- Trend analysis  
- Visualisation using bar and pie charts  
- Python libraries: Pandas, Matplotlib  

## 📈 Key Insights
- Flu peaks in winter (Dec–Jan)  
- Asthma increases in late spring and summer  
- Stomach-related illnesses remain steady  

## 🖼 Visuals
Charts are stored in the `visuals/` folder.
