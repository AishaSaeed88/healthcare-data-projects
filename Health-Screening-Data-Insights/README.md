# Health Screening Data Insights

## 📌 Overview
This project analyses anonymised school health screening data, including BMI, vision scores, and follow‑up requirements. The aim is to identify early health risks and demonstrate analytical reasoning.

## 📊 Dataset
Columns:
- ID  
- Age  
- Gender  
- BMI  
- Vision (Left/Right)  
- Screening Result  

## 🧪 Methods
- Data exploration  
- BMI categorisation  
- Visualisation using bar and pie charts  
- Python libraries: Pandas, Matplotlib  

## 📈 Key Insights
- Overweight students had higher follow‑up rates  
- Vision issues increased with age  
- Screening data supports targeted health interventions

Charts are available in the `visuals/` folder.
